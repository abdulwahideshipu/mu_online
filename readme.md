# MU Online
Introducing MU Online, a comprehensive and dynamic platform designed for students to easily register and access a wide range of courses offered by Metropolitan University. This innovative website serves as a central hub for educational opportunities, providing a seamless and user-friendly experience for learners. One of the unique features of MU Online is that it empowers anyone to upload and teach their own courses, thus becoming part of the esteemed MU Faculty. This not only broadens the scope of available courses but also encourages knowledge sharing and expertise from diverse fields. MU Online aims to significantly enhance both learning and teaching experiences, fostering a collaborative and enriching educational environment that benefits students and educators alike.

## Methodology
The MU Online platform was developed using HTML, CSS, JavaScript, and PHP for the frontend, creating an interactive and responsive user interface. The backend, coded in PHP, manages user registrations, course uploads, and database interactions. PHPMyAdmin was used for database management, ensuring efficient data storage and retrieval. This combination ensures a seamless and robust online learning environment.

## Tools Used
The MU Online project utilized several essential tools to ensure robust performance and a seamless user experience. Apache served as the web server, providing a stable and secure platform for hosting the website. MySQLServer was employed for database management, enabling efficient storage and retrieval of user and course data. Development was carried out using VS Code, a versatile and powerful coding environment for HTML, CSS, JavaScript, and PHP. Google Chrome was used as the primary web browser for testing and debugging, ensuring compatibility and smooth performance across different devices. These tools collectively contributed to the successful development and deployment of MU Online.
